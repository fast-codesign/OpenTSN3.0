本目录中各文件夹说明如下：
ipcore：      	     TSN测试仪示例工程使用的所有IP核以及其存放路径
rtl：                              TSN测试仪示例工程的器件相关代码和接口相关代码
create_project.tcl:         创建TSN测试仪示例工程的脚本文件
Makefile:                      编译TSN测试仪示例工程的Makefile文件，详细使用请见OpenTSN3.0-centralized\Hardware\doc\使用手册目录下的TSN网卡使用手册第三章节（注：TSN测试仪Makefile的操作和TSN网卡Makefile的操作相同）
TSN_FPGA_4port.sdc:   TSN测试仪示例工程的设计约束文件
